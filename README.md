# Smart Task Organizer (Python)

A simple text-based Smart Task Organizer built using Python.  
This project helps users add, view, complete, and manage daily tasks using a command-line interface.

## Features
- Add new tasks
- View all tasks
- Mark tasks as completed
- Tasks are stored persistently in a text file
- Simple and beginner-friendly Python logic

## Technologies Used
- Python
- File handling (TXT file)
- Command Line Interface (CLI)

## How to Run the Project
1. Clone the repository
2. Run the file using:
   python main.py
3. Follow on-screen options

## Project Use Case
This project demonstrates:
- Python fundamentals
- File handling
- Logic building
- Real-world task management workflow
